## Global Rule: Clarification First

Before making any major visual modifications, rewriting existing core features, or deleting previously custom-built interactive canvases (e.g., charts, plots, simulators), the agent **MUST** explicitly state all assumptions, outline the design plan, and obtain explicit user activation. 

### Why this is critical:
- Bypassing clarification can lead to incorrect alignment or the wasteful deletion of completed work.
- It preserves user-approved interactive elements like the **Interactive Workspace Simulator** and the **Value Matrix Diagram**.
